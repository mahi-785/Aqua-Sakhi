# 🌸 AquaSakhi

A cute desktop water companion that reminds you to stay hydrated.

## Features

- 💧 Water reminders
- 🌸 Animated desktop companion
- 🖥 Transparent desktop widget
- ⏰ Configurable reminder intervals
- 😴 Snooze reminders
- 📊 Daily hydration statistics

## Built With

- Python
- PySide6
- Pillow
